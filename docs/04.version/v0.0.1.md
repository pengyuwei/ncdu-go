# v0.0.1

1. [x] 实现一个简单版本的ncdu功能
2. [x] 命令行参数支持文件路径（可选），没有参数默认当前目录。
3. [x] 首先列举出目标路径所有的文件和文件夹列表并按照文件/文件夹名、占比进度条，实际容量的顺序显示每行信息，
4. [x] 然后异步检查每个文件夹的容量大小。
5. [x] 用户可以使用上下方向键来移动当前行，超过终端窗口范围则滚动展示。按右方向键进入当前选择的文件夹，按左去往上一级文件夹，进入和返回都重新刷新当前文件夹列表和容量，已经获取完容量的文件夹的容量值，内存中缓存，下次就不重复获取，按F5重新刷新当前文件夹容量信息（并清空下级缓存）
6. [x] 占比进度条为当前路径当前光标选择行所占容量/总容量的进度比
7. [x] 容量信息要自动进行翻译：小于1024字节用B为单位，依次类推按照实际大小显示为KB/MB/GB/TB，保留2位小数
8. [x] 写一个make文件，用于编译，并且按照github命名习惯生成bin的release包
